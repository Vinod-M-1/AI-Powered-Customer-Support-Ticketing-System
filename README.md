## 📖 Project Overview

**AI-Powered Customer Support Ticketing System** is a Salesforce-based CRM solution developed to streamline and automate customer support operations for TheChaiShop&Co. Built during the **SRMAP University Virtual Internship Program – Salesforce Certified Administrator with AI Agentforce Specialization**, the system manages customer, employer, and enterprise accounts efficiently while automating case classification, priority assignment, SLA tracking, and AI-generated reply suggestions.

The primary objective of this project is to reduce manual support effort, improve response consistency, and enable data-driven decision-making for support teams through Salesforce's native automation and generative AI capabilities.

---

## ✨ Features

- 🎟️ Automated customer support case creation and management
- 🤖 AI-generated suggested replies using Prompt Builder & Agentforce
- ⚙️ Automated case classification and priority assignment
- ⏱️ SLA due date tracking and compliance monitoring
- 📧 Automated email notifications on case resolution

## ▶️ How It Works

1. Sign up for a Salesforce Developer Edition account: `https://developer.salesforce.com/signup`
2. Configure standard objects (Account, Contact, Case) and custom objects (Support Category, SLA Policy, AI Feedback)
3. Build the **AI-Powered CST System** Lightning App with custom tabs
4. Implement Validation Rules and Record-Triggered Flows for automation
5. Configure Prompt Builder and Agentforce for AI Suggested Replies
6. Set up Profiles, Permission Sets, and Users for role-based access
7. Create Reports and Dashboards to monitor support operations
8. Test end-to-end: Case creation → AI reply generation → Resolution → Reporting
